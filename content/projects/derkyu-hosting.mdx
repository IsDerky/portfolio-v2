## Overview

A comprehensive hosting solution built for the gaming and developer community — offering game servers, Discord bot hosting, and custom software deployment. The goal was to eliminate the complexity of self-hosting while keeping full control over infrastructure.

## The Problem

Most hosting providers are either too generic (AWS, Hetzner) and require significant DevOps knowledge, or too specialized and overpriced (e.g., Pterodactyl-based hosts). Derkyu Hosting sits in the middle: a managed experience with sensible defaults, but full flexibility for advanced users.

## Features

- **Scalable plans** with 99% SLA and transparent pricing
- **Strategic server locations** across Europe and South America for low latency
- **24/7 technical support** with fast response times
- **Integrated code editor** and SFTP file manager — no SSH required for basic tasks
- **Automatic updates** and real-time monitoring dashboards
- **DDoS protection** included on all plans

## Tech Stack

Built with **Next.js**, **TypeScript**, and **shadcn/ui** for the control panel frontend, deployed on a custom infrastructure stack optimized for low-latency game server workloads.

## What I Learned

Managing infrastructure at scale surfaced challenges around resource isolation, billing logic, and real-time monitoring that pushed me to build tooling I hadn't originally planned. The project is ongoing and actively used by clients.
